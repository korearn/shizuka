# 静か (Shizuka) ・ Zero-Noise Scaffolding

> A minimalist, Tokyo Night inspired project scaffolder for developers who seek a digital refuge.

Shizuka (meaning *quiet* or *calm* in Japanese) is a lightweight CLI tool designed to generate clean, bloat-free project structures. No unnecessary configs, no visual noise. Just the absolute essentials to start coding in peace.

## 🌌 Aesthetics
Designed with the terminal in mind. Shizuka uses a tailored Tokyo Night color palette to blend seamlessly with a riced Linux workflow.
